        # web3 那些事

        推荐小 lin 说视频（适合小白科普）：

        1. web3 是未来，还是骗局 (opens new window) (opens new window)
        2. 关于元宇宙，你必须知道的那些事 (opens new window) (opens new window)

        # 1、什么是 web3.0

        ​ web3.0 利用分布式账本技术，构建了一个激励相容的开放式环境，web3.0 赋予了用于真正的数据自主权，个人信息将成为用户掌握的数据资产，用户可以在数据流转和交易中真正获益，使自己的数据不再是互联网平台的免费资源。

        web1.0（1995-2005） web2.0（2005-2014） web3.0（2014-？）
        交互方式 读 读+写 读+写+拥有
        控制权 网站提供内容
        用户阅读内容 用户可生成内容
        与他人和网站交互 （数字）身份、资产和数据回归个人
        账户系统 邮箱 手机号码 私钥/助记词/智能钱包
        组织形式 公司 平台 DAO/网络
        生产关系 B 端平台的广告收入 平台收摊位费和广告费 按照工作量计算
        基础设施 个人 PC 手机&云 区块链
        案例 搜狐、网易、新浪等
        门户网站 微信、微博、小红书等
        社交平台 uniswap、opensea 等
        各类 Dapp（去中心化 app）

        # 2、web3.0 的特点

        web3.0 特性其实是跟着区块链技术走的，因为底层是基于区块链技术

        • 无需准人
        • 全球化
        • 匿名性
        • server-less：从账本、前端、后端到业务逻辑
        • 自成一体的支付和账户系统

        # 3、web3.0 的应用方向

        • DeFi（去中心化金融）
        • NFT
        • X to Earn：X 代表了 everything，如走路赚钱（move to Earn）、游戏赚钱（Play to Earn）

        # 4、产业全景图

        理想的 web3 项目=资产（加密货币 NFT）+数据（DID）+社区治理（DAO）+商业逻辑（Defi、socialFi、GameFi、x to Earn...）

        产业全景图如下：

        • DAPP

          • 消费端：socialFi、web2 平台、NFT 市场（OpenSea）、音乐、影视/视频、GameFi
          • 产业端：DeFi（稳定币、借贷、DEX、保险、聚合理财）、CeFi、加密钱包、数字营销、搜索引擎

        • 基础设施

          • 身份/DID：ENS、civic、brightID、Spruce
          • 社区治理：boardroom、aracon、snapshot
          • 平台开发：forte、skymavis、immutable、Dapper
          • 数据分析：Chainalysis、Dune Analytics
          • 安全科技：OpenZeppelin、CERTIC
          • 存储：Filecoin、safe Network、storj

        • 底层技术

          • 数据传输层、物理层
          • Layer 0&1 底层链：公链、扩容/分片/跨链
          • Layer2 扩容方案：侧链、状态通道、Optimistic Rollup（ORU 乐观汇总）、Validium、Plasma、Arbitrum Rollup、Polygon、ZK Rollup

        # 1、底层技术

        • Layer3 自定义扩容：L3 是个专网的概率，比如专门提供隐私计算的算法，供别人使用
        • Layer2 链下扩容方案：高速增长，为 web3 扩容打下基础
        • Layer1 公链以及链上扩容方案：目前公链目前近 1000 条；链上扩容包括：更改共识机制，DAG 技术、分片等，以太坊第一公链
        • Layer0 区块链底层协议：类似 PC 和移动端的操作系统，支持公链搭建和跨链

        # 2、基础设施

        • DID 受市场需求驱动，对钱包功能开放

        • DAO 以代币和智能合约达成去中心化

          • web2 传统自上而下的组织架构
          • web3 去中心化自治组织 DAO
            • 以智能合约形式运行和自动执行社区管理条例
            • 以代币作为权利分配和激励机制的媒介
            • 不存在中心化主体，核心决策由社区选拔

        # 3、DAPP

        • 数字营销业务雏形已现，但风险漏洞较多
        • 数字营销从利益分配实现 traffic2earn
        • socialFi 将内容控制权还给用户
        • Axie Infinity 引领 GameFi 赛道的 P2E 模式
        • GameFi（X2E）是新叙事范式的试验田
